# Velvet Bean | Luxury Coffee Shop Website

**Velvet Bean** is a modern, static portfolio website designed for a high-end coffee shop concept. The design merges "Industrial Luxury" with "Digital Brutalism," featuring asymmetrical grids, bold typography, and smooth scroll animations.

This project was built using pure, vanilla technologies to ensure high performance and zero dependencies.

* **HTML5:** Semantic structure for better SEO and accessibility.
* **CSS3:** Custom styling without frameworks (No Bootstrap/Tailwind).
    * CSS Grid & Flexbox for layout.
    * CSS Variables for theming.
    * Glassmorphism effects.
    * Keyframe animations.
* **Vanilla JavaScript:**
    * Custom cursor logic.
    * Intersection Observer for scroll-triggered animations.
    * Horizontal scroll logic for the gallery section.

## ‚ú® Key Features
* **Industrial Aesthetic:** Dark mode design with "Burnt Orange" accents and brutalist typography (`Syne` & `Roboto Mono`).
* **Bento Grid Menu:** A modern, grid-style menu layout with hover reveal effects.
* **Horizontal Scroll Gallery:** A unique "Vibe" section that scrolls horizontally while the user scrolls down vertically.
* **Custom Interactions:** Includes a magnetic custom cursor (desktop) and scroll-triggered fade-in animations.
* **Fully Responsive:** Optimized for Android/Mobile with a custom hamburger menu and native touch swiping.
